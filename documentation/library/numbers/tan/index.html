tan

calculate the tangent of given angle

Parameters

tan angle :integer :floating

Returns

  • :floating

Examples

copy
print tan 0             ; 0.0
print tan 0.3           ; 0.3093362496096232
print tan 1.0           ; 1.557407724654902
