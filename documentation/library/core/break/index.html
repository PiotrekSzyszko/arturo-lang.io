break

break out of current block or loop

Parameters

break

Returns

  • :block

Examples

copy
loop 1..5 'x [
    print ["x:" x]
    if x=3 -> break
    print "after check"
]
print "after loop"

; x: 1
; after check
; x: 2
; after check
; x: 3
; after loop
