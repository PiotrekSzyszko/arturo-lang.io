false?

returns true if given value is false; otherwise, it returns false

Parameters

false? value :any

Returns

  • :logical

Examples

copy
print false? 1 = 2          ; true
print false? 1 <> 2         ; false
print false? odd? 2         ; true

print false? [1 2 3]        ; false
