lower

convert given string to lowercase

Parameters

lower string :string :literal

Returns

  • :string
  • :nothing

Examples

print lower "hello World, 你好!"      ; "hello world, 你好!"

str: "hello World, 你好!"
lower 'str                           ; str: "hello world, 你好!"
