take

keep firstof elements from given collection and return the remaining ones

Parameters

take collection :string :literal :block
     number :integer

Returns

  • :string
  • :block
  • :nothing

Examples

copy
str: take "some text" 5
print str                     ; some

arr: 1..10
take 'arr 3                   ; arr: [1 2 3]
